# Luna Math Adventure

Juego educativo Flutter para practicar conteo, sumas y restas de forma visual, amable y bilingue.

## Objetivo del MVP

- Onboarding basico: idioma, nombre de la nina y unicornio/unicornia.
- Navegacion inicial entre mapa, juego, practica, ayudas, recompensas y ajustes.
- Motor de ejercicios preparado para niveles progresivos.
- TTS local en espanol y catalan.
- Audio local para efectos y musica.
- Datos configurables en JSON.
