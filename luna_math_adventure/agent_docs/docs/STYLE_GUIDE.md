# Style Guide

## Personalidad visual

La app debe sentirse:

- mágica.
- suave.
- amable.
- infantil.
- alegre.
- no competitiva.
- segura.
- clara.

## Paleta inicial

```text
Rosa pastel:        #F8A9D8
Lila suave:         #C7A4FF
Azul cielo:         #A9DDFB
Menta suave:        #BDF7D1
Amarillo estrella:  #FFE680
Blanco cálido:      #FFF8F0
Morado texto:       #5C3A7D
Verde acierto:      #7ED957
Naranja pista:      #FFB84D
```

## Uso

### Fondo

- azul cielo.
- lila suave.
- degradados ligeros.
- fondos ilustrados.

### Tarjetas

- blanco cálido.
- bordes redondeados.
- sombra suave.
- mucho padding.

### Botones

- grandes.
- redondeados.
- texto claro.
- estados visuales evidentes.

### Aciertos

- verde suave.
- sparkle.
- animación positiva.
- sonido amable.

### Errores

- no usar rojo fuerte.
- usar naranja suave o lila.
- mensaje amable.
- ofrecer pista.

## Tipografía

Usar una fuente infantil, redondeada y legible.

Requisitos:

- buena lectura.
- soporte español y catalán.
- tildes.
- ç.
- apóstrofes.
- números claros.

## Botones

Tamaños grandes para uso infantil.

Ejemplo:

```text
Altura mínima: 56-64 px
Bordes: 20-28 px
Padding generoso
```

## Iconos

Iconos funcionales desde librerías:

- altavoz.
- ajustes.
- repetir.
- pista.
- home.
- continuar.
- candado.

Evitar generar iconos funcionales con IA salvo casos especiales.

## Animaciones

Usar animaciones cortas.

Buenas:

- bounce.
- fade.
- sparkle.
- confeti breve.
- aparición suave.

Evitar:

- animaciones largas.
- estímulos excesivos.
- parpadeos intensos.
- castigos visuales.

## Tono de texto

Mensajes cortos, positivos y claros.

Correcto:

```text
¡Genial!
¡Lo has conseguido!
¡Muy bien!
```

Error:

```text
Casi. Vamos a contarlo juntas.
Buena prueba. ¿Miramos una pista?
```

No usar:

```text
Mal.
Incorrecto.
Has fallado.
```

## Layout de pantalla de ejercicio

Elementos:

```text
Personaje
Enunciado
Botón altavoz
Visualización matemática
Opciones de respuesta
Botón pista
Progreso del nivel
```

Prioridad:

1. Enunciado.
2. Visualización.
3. Opciones.
4. Pista.
5. Recompensa/progreso.

## Accesibilidad

- botones grandes.
- buen contraste.
- TTS disponible.
- no depender solo de color.
- texto claro.
- evitar saturación visual.
