# Audio Cues

Formato elegido para SFX: `.wav`.

Los archivos pueden no existir al principio. El codigo ya referencia estas rutas y usa fallback del sistema mientras se van incorporando.

## UI

- `assets/audio/sfx/ui/tap_01.wav`
  - cualquier seleccion normal.
  - botones de home.
  - elegir idioma.
  - elegir personaje.
  - crear/cambiar perfil.

- `assets/audio/sfx/ui/back_01.wav`
  - volver atras.
  - cerrar panel o modal.

## Feedback

- `assets/audio/sfx/feedback/correct_01.wav`
  - respuesta correcta.
  - confirmar accion importante completada.

- `assets/audio/sfx/feedback/wrong_soft_01.wav`
  - respuesta incorrecta amable.
  - nunca debe sonar a castigo.

- `assets/audio/sfx/feedback/hint_open_01.wav`
  - abrir pista.
  - mostrar ayuda contextual.

## Recompensas

- `assets/audio/sfx/rewards/sparkle_01.wav`
  - microcelebracion.
  - pequenas estrellas o sparkles visuales.

- `assets/audio/sfx/rewards/reward_unlock_01.wav`
  - desbloquear accesorio, pegatina o insignia.

- `assets/audio/sfx/rewards/level_complete_01.wav`
  - completar nivel.

- `assets/audio/sfx/rewards/sticker_collected_01.wav`
  - conseguir pegatina.

## Musica

La musica puede quedarse como `.mp3` mas adelante:

- `assets/audio/music/happy_magic_loop_01.mp3`
- `assets/audio/music/calm_rainbow_loop_01.mp3`
