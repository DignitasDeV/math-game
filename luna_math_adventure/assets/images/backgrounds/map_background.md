# Map background

Generar dos imagenes de mapa general, manteniendo el mismo estilo visual y la misma identidad.

## Entregable 1: fondo de pantalla

Nombre recomendado: `map_background_screen.webp`

Resolucion: `1536x2048 px`

Relacion de aspecto: `3:4`

Uso: fondo vertical para tablet en la pantalla de mapa de mundos y seleccion de niveles.

Prompt: mapa magico claro y suave, con caminos entre zonas: Bosque de Corazones, Lago de Estrellas, Camino Arcoiris, Castillo de Cristal, Torre Magica y Jardin de Flores. Composicion vertical para tablet, con zonas limpias donde Flutter pueda colocar tarjetas o nodos encima. Debe sugerir mundo conectado sin dibujar textos ni botones.

## Entregable 2: card de mapa general

Nombre recomendado: `map_background_card.webp`

Resolucion: `1536x2304 px`

Relacion de aspecto: `2:3`

Uso: tarjeta vertical o portada de "Mapa" si se necesita una card general.

Prompt: version vertical tipo portada del mapa magico, con caminos suaves y pequenos indicios visuales de cada mundo. El foco debe estar centrado. Dejar la zona inferior limpia y ligeramente mas oscura o simple para que se lea texto blanco encima.

## Estilo

Ilustracion 2D infantil, colores pastel, lectura clara en tablet vertical, sin texto, sin logos.

## Evitar

Texto integrado, perspectiva demasiado compleja, nodos ya dibujados si luego se renderizan en Flutter, exceso de detalle, elementos importantes pegados a los bordes.
