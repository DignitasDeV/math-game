# home_action_map

Nombres finales:

- `home_action_map_es.webp`
- `home_action_map_ca.webp`

Uso: imagen de card para la accion "Mapa" en la home.

Resolucion recomendada: `1536x768 px`

Relacion de aspecto: `2:1`

Texto integrado:

- ES: `Mapa`
- CA: `Mapa`

Prompt: tarjeta horizontal infantil magica para abrir el mapa, con camino suave, brillos pequenos, indicios de mundos magicos y composicion limpia. Incluir el texto indicado grande, centrado y muy legible. Sin logos.
